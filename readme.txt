Комментарии к программе: 
 - Я писал на работе, где стоит Windows 7. Поэтому писал на python 3.8, более новые версии на него не ставятся. А так как конструкция match...case появилась только в python 3.10, то воспользовался менее красивой конструкцией if..elif
- Внешние библиотеки были запрещены, поэтому не использовал библиотеку json, с  которой было бы удобнее
- Еще удобнее было бы использовать базу данных, но по правилам можно было только текстовый файл.
- Тесты из фйла tests.py будут работать только на пустом файле library.py из-за того, что данные в нем всё время меняются, а ID всегда разный.
- Только для проведения тестов добавил return в методы класса Books. Так как всё итак выводится на экран консоли, то можно было и без них.


Работа программы:
 - Файл library.txt должен лежать в одном каталоге с программой.
 - При запуске отобразится меню библиотеки. 
 - Выбирайте нужный пункт вводя его номер, следуйте указаниям программы.
 - Не вносите руками коррективы в файл library.txt. При ошибке, библиотека обнулится.


Как устроена библиотека:
Каждая книга в библиотеке имеет ID (назначается программой), Название, Автора, Год выхода и статус(изначально назначается программой как "в наличии")
Можно добавлять свои книги, вводя Название, автора и год выхода - (пункт 1)
Удалять книги из библиотеки, вводя ID книги - (пункт 2) 
Искать книги по части их названия, автору или году выхода (не зависимо от регистра) - (пункт 3)
Вывести на экран все книги библиотеки с их ID и статусами - (пункт 4)
Изменить статус книги на "выдана" или "в наличии" по ID книги - (пунки 5)
